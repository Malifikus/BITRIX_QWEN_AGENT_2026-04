# Модуль "Расширение календаря" для Битрикс24

## Описание
Модуль добавляет функционал обязательного выбора типа события календаря и привязки компании из CRM для встреч.

## Функционал
- Обязательный выбор типа события: Встреча, Собеседование, Созвон, Другое
- При выборе типа "Встреча" появляется обязательное поле выбора компании из CRM
- Данные сохраняются в описании события в формате JSON
- Данные восстанавливаются при редактировании события
- Валидация на клиенте и сервере

## Установка
1. Скопируйте папку `custom_calendar` в `/local/modules/` вашего сайта
2. В административной панели перейдите: **Marketplace → Установленные решения**
3. Найдите модуль **"Расширение календаря"** (custom_calendar) и нажмите **"Установить"**
4. Следуйте инструкциям мастера установки
5. После установки модуль автоматически зарегистрирует обработчики событий календаря

## Использование
После установки модуля:
1. Откройте календарь в Битрикс24
2. Создайте или отредактируйте событие
3. Выберите тип события из выпадающего списка:
   - **Встреча** — требуется выбор компании из CRM
   - **Собеседование**
   - **Созвон**
   - **Другое**
4. Если выбран тип "Встреча" — выберите компанию из CRM через диалог выбора
5. Сохраните событие

Данные сохраняются в поле DESCRIPTION события в виде JSON-маркера:
```html
<!--CUSTOM_CALENDAR_DATA-->{"EVENT_TYPE":"meeting","COMPANY_ID":123}</!--CUSTOM_CALENDAR_DATA-->
```

## Удаление
1. В административной панели: **Marketplace → Установленные решения**
2. Найдите модуль **"Расширение календаря"**
3. Нажмите **"Удалить"**
4. Подтвердите удаление (можно сохранить данные в событиях календаря)

При удалении:
- Отключаются обработчики событий
- Удаляются файлы компонента из `/local/components/custom/calendar.event.edit/`
- Удаляются JS-файлы из `/local/js/calendar/`

## Требования
- Битрикс24 Коробочная версия
- PHP 7.0+
- Bitrix Framework 17.0+
- Модуль CRM (для выбора компаний)

## Примечания
- Модуль использует стандартные события календаря: `OnBeforeCalendarEventAdd`, `OnBeforeCalendarEventUpdate`
- Для работы выбора компании необходим современный интерфейс Битрикс24 (BX.CrmEntitySelectorDialog)
- При редактировании события данные автоматически загружаются из DESCRIPTION
