# Корпоративный Университет - LMS платформа для Битрикс24

Приложение для управления корпоративным обучением с интеграцией в Битрикс24 через VibeCode API.

## 🚀 Возможности

- **Каталог курсов** - просмотр и запись на учебные курсы
- **Прогресс обучения** - отслеживание прогресса сотрудников
- **Статистика** - дашборды по обучению и продажам (CRM)
- **Сотрудники** - профиль обучения каждого сотрудника
- **Вебинары** - расписание онлайн-встреч
- **Уведомления** - отправка уведомлений через Битрикс24
- **Задачи** - создание учебных задач

## 🔧 Установка

1. Установите зависимости:
```bash

```

2. Настройте API ключ в `config/settings.py`:
```python

```

3. Запустите приложение:
```bash
cd corporate-university
python src/app.py
```

4. Откройте браузер: http://localhost:5000

## 📊 API Endpoints

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/` | GET | Главная страница |
| `/courses` | GET | Каталог курсов |
| `/course/<id>` | GET | Детали курса |
| `/employees` | GET | Список сотрудников |
| `/employee/<id>` | GET | Профиль сотрудника |
| `/statistics` | GET | Статистика обучения |
| `/webinars` | GET | Вебинары |
| `/api/stats` | GET | JSON статистика |
| `/api/courses` | GET | JSON курсы |
| `/api/enroll` | POST | Запись на курс |
| `/api/notify` | POST | Отправка уведомления |
| `/health` | GET | Проверка здоровья |

## 🔌 Интеграция с Битрикс24

Приложение использует VibeCode API для работы с:
- **Пользователи** - получение списка сотрудников
- **Задачи** - создание и управление учебными задачами
- **Отделы** - структура компании
- **CRM** - контакты, компании, сделки
- **Каталог** - хранение курсов и уроков
- **Уведомления** - пуш-уведомления пользователям
- **Календарь** - вебинары и встречи

## 🎨 Технологии

- **Backend**: Python 3.x, Flask
- **API Client**: requests
- **Frontend**: HTML5, CSS3, Jinja2 templates
- **Интеграция**: VibeCode API (Битрикс24)
